TFRecord를 이용해 데이터를 불러오기 위해 필요한 연습코드

TFRecord 및 멀티 쓰레드를 함께 사용해야함.

연습 순서는 아래와 같음.
1. tfrecord_read.py		: tfrecord 연습
2. tfrecord_write.py		: tfrecord 연습
3. queue_basic.py		: 기본 queue 연습
4. multi_thread.py		: multi thread에  대한 기본 연습
5. coordinator.py		: thread들을 다룰 수 있는 coordinator를 함께 이용하여 queue 쌓음.  
6. main.py : 위 연습코드들을 통합.
